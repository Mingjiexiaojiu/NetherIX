# NetherIX

> *"来自冥界第九层的守望者"*

**NetherIX**（发音：[ˈneðər aɪ-eks]）是一款桌面宠物应用。

- **Nether** → 源自古英语，意为"下方、深处"，完美对应「冥」界的幽深意象
- **IX** → 罗马数字 9，对应「九」，复古而神秘
- **NIX** → 自然形成的 Logo 缩写，也是北欧神话中水域精灵的名字 🖤

他的中文名叫做**冥九灵**，你也可以直接叫他 **NIX**。

---

NIX 栖居在你的屏幕一角，来自冥界第九层，沉默、神秘，却始终陪伴。

## 功能

- **桌面精灵** -- 透明无边框小精灵，自由行走于桌面，支持拖拽、自主行为（闲逛、坐下、睡觉）
- **AI 大脑** -- 接入 LLM（通义千问 / GPT / DeepSeek），支持自然语言对话与意图识别
- **自动化操控** -- 鼠标键盘控制、应用启动/窗口管理、系统设置调整
- **文件处理** -- 文件 CRUD、搜索、批量重命名、按类型自动整理
- **技能系统** -- 插件式架构，内置计算器/翻译/天气/搜索/提醒/文件整理，支持自定义技能
- **全局唤醒** -- `Ctrl+Space` 快捷键呼出输入框，随时与 NIX 对话

## 快速开始

```bash
pip install -r requirements.txt
python main.py
```

在 `config.yaml` 中配置你的 AI API Key 即可启用完整的智能助手功能。
